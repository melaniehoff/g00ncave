# g00n cave

![](g00ncave.jpg)

### Dependencies

* go
* [reserve](https://github.com/s4y/reserve)

### Installing

* install go and add the go path to bash profile

```
export GOPATH=$HOME/go
export PATH=$GOPATH/bin:$PATH
```
### Running

```
reserve -http :8080
```
